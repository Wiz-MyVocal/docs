---
title: "Get TTS History Url"
api: "POST https://api.myvocal.ai/sound_clone/api/v1/tts/url"
description: "Get tts history items' audio url ."
---
### Header
<ParamField header="accessKey"   type="string" required>
  You can copy your api-key on https://www.myvocal.ai/profile after you have subscribed to Pro or Business Plan.
</ParamField>

### Body
<ParamField body="ttsIds" type="array" required>
 You can provide up to 1,000 Ids at a time.
</ParamField>

### Response

<ResponseField name="code" type="number">
  Indicates whether the call was successful. 1 if successful.
</ResponseField>

<ResponseField name="message" type="string" >
  When an exception occurs, specific exception information will be provided in the message.
</ResponseField>

<ResponseField name="data" type="array">
  <Expandable title="Toggle object">
   
  <ResponseField name="ttsId" type="string">
      Your tts id.
  </ResponseField>
   <ResponseField name="ttsUrl" type="string">
     The audio file url corresponding to ttsId.
  </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash Example Request
```bash Example Request
curl --location --request POST 'https://api.myvocal.ai/sound_clone/api/v1/tts/url' \
--header 'Content-Type: application/json' \
--header 'accessKey: xxxx' \
--data '{
    "ttsIds":[
        "0",
        "1"
    ]
}'
```

</RequestExample>
